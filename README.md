

# Unsplash

This android application works with binlist.net api



## Authors

- [@blblblblblblblblbl](https://github.com/blblblblblblblblbl)


## Features

- search
- history
- detailed info
- phone in external app
- open map in external app
- open website in external app



## tech stack
- Android studio
- Kotlin
- Clean Architecture
- multi-module app
- Coroutines 
- Flow
- OOP
- SOLID
- binlist.net api
- Retrofit2
- OkHttpClient
- Paging library3
- Jetpack Compose
- Jetpack Navigation
- Material Design 3
- Material You
- Room
- Remote mediator
- Hilt
